Allows you to display the poster avatar for posts, comments and in the admin pages.

== Description ==

Poster Avatar allows you to display the poster avatar in different places :
1. Main pages of the blog
2. Recent comments widget
3. All admin pages like posts, pages, medias, links etc...

== Installation ==

1. Upload `posteravatar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where are the avatars getting from ? =

Poster Avatar request Gravatar database.
